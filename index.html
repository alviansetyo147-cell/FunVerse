FUNVERSE

Belajar Fungsi Invers dengan Mudah dan Menyenangkan

📚 Pengertian Fungsi Invers

Fungsi invers adalah fungsi yang membalik proses suatu fungsi sehingga output dapat dikembalikan menjadi input semula.

Fungsi Asal
3 → 8 (+5)

Fungsi Invers
8 → 3 (-5)

Bayangkan seperti mengunci dan membuka kunci pintu. Keduanya merupakan proses yang saling berlawanan.

📖 Syarat Fungsi Memiliki Invers

  • Fungsi harus bersifat satu-satu (injektif).
  • Fungsi harus bersifat bijektif agar semua nilai memiliki pasangan.

🧮 Langkah Menentukan Fungsi Invers

  1. Misalkan y = f(x)
  2. Tukar x dan y
  3. Cari nilai y
  4. Tulis sebagai f⁻¹(x)

📝 Contoh Soal

f(x)=2x+4

y = 2x + 4
x = 2y + 4
x - 4 = 2y
y = (x - 4)/2

f⁻¹(x) = (x - 4)/2

⚡ Ringkasan Rumus Cepat

  • x + a → x − a
  • x − a → x + a
  • ax → x/a
  • x/a → ax

📌 Kesimpulan

  • Fungsi invers membalik fungsi asal.
  • Tidak semua fungsi memiliki invers.
  • Cari invers dengan menukar x dan y lalu mencari y.
  • Latihan yang rutin akan mempermudah memahami materi ini.
